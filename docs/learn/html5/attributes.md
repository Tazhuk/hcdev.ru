# –ê—Ç—Ä–∏–±—É—Ç—ã

<big>–£–∑–Ω–∞–π—Ç–µ –æ —Ä–∞–∑–ª–∏—á–Ω—ã—Ö –≥–ª–æ–±–∞–ª—å–Ω—ã—Ö –∞—Ç—Ä–∏–±—É—Ç–∞—Ö, –∞ —Ç–∞–∫–∂–µ –æ–± –∞—Ç—Ä–∏–±—É—Ç–∞—Ö, —Ö–∞—Ä–∞–∫—Ç–µ—Ä–Ω—ã—Ö –¥–ª—è –∫–æ–Ω–∫—Ä–µ—Ç–Ω—ã—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ HTML.</big>

–ï—Å–ª–∏ –∑–Ω–∞—á–µ–Ω–∏–µ –∞—Ç—Ä–∏–±—É—Ç–∞ —Å–æ–¥–µ—Ä–∂–∏—Ç –ø—Ä–æ–±–µ–ª –∏–ª–∏ —Å–ø–µ—Ü–∏–∞–ª—å–Ω—ã–µ —Å–∏–º–≤–æ–ª—ã, —Ç–æ –∑–Ω–∞—á–µ–Ω–∏–µ –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å –∑–∞–∫–ª—é—á–µ–Ω–æ –≤ –∫–∞–≤—ã—á–∫–∏. –ü–æ —ç—Ç–æ–π –ø—Ä–∏—á–∏–Ω–µ, –∞ —Ç–∞–∫–∂–µ –¥–ª—è –ø–æ–≤—ã—à–µ–Ω–∏—è —Ä–∞–∑–±–æ—Ä—á–∏–≤–æ—Å—Ç–∏, –≤—Å–µ–≥–¥–∞ —Ä–µ–∫–æ–º–µ–Ω–¥—É–µ—Ç—Å—è –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å –∫–∞–≤—ã—á–∫–∏.

–•–æ—Ç—è HTML –Ω–µ —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª–µ–Ω –∫ —Ä–µ–≥–∏—Å—Ç—Ä—É, –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è –∞—Ç—Ä–∏–±—É—Ç–æ–≤ —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω—ã –∫ –Ω–µ–º—É. –ó–Ω–∞—á–µ–Ω–∏—è, —è–≤–ª—è—é—â–∏–µ—Å—è —á–∞—Å—Ç—å—é —Å–ø–µ—Ü–∏—Ñ–∏–∫–∞—Ü–∏–∏ HTML, –Ω–µ—á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω—ã –∫ —Ä–µ–≥–∏—Å—Ç—Ä—É. –û–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è —Å—Ç—Ä–æ–∫, —Ç–∞–∫–∏–µ –∫–∞–∫ –∏–º–µ–Ω–∞ –∫–ª–∞—Å—Å–æ–≤ –∏ `id`, —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω—ã –∫ —Ä–µ–≥–∏—Å—Ç—Ä—É. –ï—Å–ª–∏ –∑–Ω–∞—á–µ–Ω–∏–µ –∞—Ç—Ä–∏–±—É—Ç–∞ —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ –∫ —Ä–µ–≥–∏—Å—Ç—Ä—É –≤ HTML, —Ç–æ –æ–Ω–æ —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ –∫ —Ä–µ–≥–∏—Å—Ç—Ä—É –ø—Ä–∏ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–∏ –≤ –∫–∞—á–µ—Å—Ç–≤–µ —á–∞—Å—Ç–∏ [—Å–µ–ª–µ–∫—Ç–æ—Ä–∞ –∞—Ç—Ä–∏–±—É—Ç–æ–≤](https://developer.mozilla.org/docs/Web/CSS/Attribute_selectors) –≤ CSS –∏ JavaScript; –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ –æ–Ω–æ –Ω–µ —á—É–≤—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ –∫ —Ä–µ–≥–∏—Å—Ç—Ä—É.

## –ë—É–ª–µ–≤—ã –∞—Ç—Ä–∏–±—É—Ç—ã {#boolean-attributes}

–ï—Å–ª–∏ –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É–µ—Ç –±—É–ª–µ–≤—Å–∫–∏–π –∞—Ç—Ä–∏–±—É—Ç, —Ç–æ –æ–Ω –≤—Å–µ–≥–¥–∞ –∏—Å—Ç–∏–Ω–µ–Ω. –ö –±—É–ª–µ–≤—ã–º –∞—Ç—Ä–∏–±—É—Ç–∞–º –æ—Ç–Ω–æ—Å—è—Ç—Å—è `autofocus`, `inert`, `checked`, `disabled`, `required`, `reversed`, `allowfullscreen`, `default`, `loop`, `autoplay`, `controls`, `muted`, `readonly`, `multiple,` –∏ `selected`. –ï—Å–ª–∏ –æ–¥–∏–Ω (–∏–ª–∏ –Ω–µ—Å–∫–æ–ª—å–∫–æ) –∏–∑ —ç—Ç–∏—Ö –∞—Ç—Ä–∏–±—É—Ç–æ–≤ –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É–µ—Ç, —Ç–æ —ç–ª–µ–º–µ–Ω—Ç —è–≤–ª—è–µ—Ç—Å—è –æ—Ç–∫–ª—é—á–µ–Ω–Ω—ã–º, –æ–±—è–∑–∞—Ç–µ–ª—å–Ω—ã–º, –¥–æ—Å—Ç—É–ø–Ω—ã–º –¥–ª—è —á—Ç–µ–Ω–∏—è –∏ —Ç. –¥. –ï—Å–ª–∏ –Ω–µ –ø—Ä–∏—Å—É—Ç—Å—Ç–≤—É–µ—Ç, —Ç–æ –Ω–µ—Ç.

–ë—É–ª–µ–≤—ã –∑–Ω–∞—á–µ–Ω–∏—è –º–æ–≥—É—Ç –±—ã—Ç—å –ª–∏–±–æ –æ–ø—É—â–µ–Ω—ã, –ª–∏–±–æ —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω—ã –≤ –ø—É—Å—Ç—É—é —Å—Ç—Ä–æ–∫—É, –ª–∏–±–æ —è–≤–ª—è—Ç—å—Å—è –∏–º–µ–Ω–µ–º –∞—Ç—Ä–∏–±—É—Ç–∞; –ø—Ä–∏ —ç—Ç–æ–º –∑–Ω–∞—á–µ–Ω–∏–µ –Ω–µ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ –¥–æ–ª–∂–Ω–æ –±—ã—Ç—å —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω–æ –≤ —Å—Ç—Ä–æ–∫—É `true`. –í—Å–µ –∑–Ω–∞—á–µ–Ω–∏—è, –≤–∫–ª—é—á–∞—è `true`, `false` –∏ `üòÄ`, –¥–∞–∂–µ –µ—Å–ª–∏ –æ–Ω–∏ –Ω–µ–¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω—ã, –ø—Ä–∏–º—É—Ç –∑–Ω–∞—á–µ–Ω–∏–µ `true`.

–≠—Ç–∏ —Ç—Ä–∏ —Ç–µ–≥–∞ —ç–∫–≤–∏–≤–∞–ª–µ–Ω—Ç–Ω—ã:


–ï—Å–ª–∏ –∑–Ω–∞—á–µ–Ω–∏–µ –∞—Ç—Ä–∏–±—É—Ç–∞ —Ä–∞–≤–Ω–æ `false`, –æ–ø—É—Å—Ç–∏—Ç–µ –∞—Ç—Ä–∏–±—É—Ç. –ï—Å–ª–∏ –∑–Ω–∞—á–µ–Ω–∏–µ –∞—Ç—Ä–∏–±—É—Ç–∞ –∏—Å—Ç–∏–Ω–Ω–æ, –≤–∫–ª—é—á–∏—Ç–µ –∞—Ç—Ä–∏–±—É—Ç, –Ω–æ –Ω–µ —É–∫–∞–∑—ã–≤–∞–π—Ç–µ –µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏–µ. –ù–∞–ø—Ä–∏–º–µ—Ä, `required="required"` –Ω–µ —è–≤–ª—è–µ—Ç—Å—è –¥–æ–ø—É—Å—Ç–∏–º—ã–º –∑–Ω–∞—á–µ–Ω–∏–µ–º –≤ HTML; –Ω–æ –ø–æ—Å–∫–æ–ª—å–∫—É `required` —è–≤–ª—è–µ—Ç—Å—è –±—É–ª–µ–≤–æ–π —Ñ—É–Ω–∫—Ü–∏–µ–π, –Ω–µ–¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è –ø—Ä–∏–≤–æ–¥—è—Ç—Å—è –∫ true. –ù–æ –ø–æ—Å–∫–æ–ª—å–∫—É –Ω–µ–¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω—ã–µ –ø–µ—Ä–µ—á–∏—Å–ª–∏–º—ã–µ –∞—Ç—Ä–∏–±—É—Ç—ã –Ω–µ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ –ø—Ä–∏–≤–æ–¥—è—Ç –∫ —Ç–æ–º—É –∂–µ –∑–Ω–∞—á–µ–Ω–∏—é, —á—Ç–æ –∏ –æ—Ç—Å—É—Ç—Å—Ç–≤—É—é—â–∏–µ –∑–Ω–∞—á–µ–Ω–∏—è, –ø—Ä–æ—â–µ –≤–∑—è—Ç—å –∑–∞ –ø—Ä–∏–≤—ã—á–∫—É –æ–ø—É—Å–∫–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏—è, —á–µ–º –∑–∞–ø–æ–º–∏–Ω–∞—Ç—å, –∫–∞–∫–∏–µ –∞—Ç—Ä–∏–±—É—Ç—ã —è–≤–ª—è—é—Ç—Å—è –±—É–ª–µ–≤—ã–º–∏, –∞ –∫–∞–∫–∏–µ –ø–µ—Ä–µ—á–∏—Å–ª–∏–º—ã–º–∏ –∏ –ø–æ—Ç–µ–Ω—Ü–∏–∞–ª—å–Ω–æ –º–æ–≥—É—Ç –¥–∞–≤–∞—Ç—å –Ω–µ–¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ–µ –∑–Ω–∞—á–µ–Ω–∏–µ.

–ü—Ä–∏ –ø–µ—Ä–µ–∫–ª—é—á–µ–Ω–∏–∏ –º–µ–∂–¥—É `true` –∏ `false` –¥–æ–±–∞–≤–ª—è–π—Ç–µ –∏ —É–¥–∞–ª—è–π—Ç–µ –∞—Ç—Ä–∏–±—É—Ç —Å –ø–æ–º–æ—â—å—é JavaScript, –∞ –Ω–µ –ø–µ—Ä–µ–∫–ª—é—á–∞—è –µ–≥–æ –∑–Ω–∞—á–µ–Ω–∏–µ.

–û–±—Ä–∞—Ç–∏—Ç–µ –≤–Ω–∏–º–∞–Ω–∏–µ, —á—Ç–æ –≤ —è–∑—ã–∫–∞—Ö XML, —Ç–∞–∫–∏—Ö –∫–∞–∫ SVG, –≤—Å–µ –∞—Ç—Ä–∏–±—É—Ç—ã –¥–æ–ª–∂–Ω—ã —Å–æ–¥–µ—Ä–∂–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ, –≤ —Ç–æ–º —á–∏—Å–ª–µ –∏ –±—É–ª–µ–≤—ã –∞—Ç—Ä–∏–±—É—Ç—ã.

## –ü–µ—Ä–µ—á–∏—Å–ª–∏–º—ã–µ –∞—Ç—Ä–∏–±—É—Ç—ã

## –ì–ª–æ–±–∞–ª—å–Ω—ã–µ –∞—Ç—Ä–∏–±—É—Ç—ã

–ì–ª–æ–±–∞–ª—å–Ω—ã–µ –∞—Ç—Ä–∏–±—É—Ç—ã ‚Äî —ç—Ç–æ –∞—Ç—Ä–∏–±—É—Ç—ã, –∫–æ—Ç–æ—Ä—ã–µ –º–æ–≥—É—Ç –±—ã—Ç—å —É—Å—Ç–∞–Ω–æ–≤–ª–µ–Ω—ã –Ω–∞ –ª—é–±–æ–º —ç–ª–µ–º–µ–Ω—Ç–µ HTML, –≤–∫–ª—é—á–∞—è —ç–ª–µ–º–µ–Ω—Ç—ã –≤ `<head>`. –°—É—â–µ—Å—Ç–≤—É–µ—Ç –±–æ–ª–µ–µ [30 –≥–ª–æ–±–∞–ª—å–Ω—ã—Ö –∞—Ç—Ä–∏–±—É—Ç–æ–≤](../../html/uni-attr.md). –•–æ—Ç—è —Ç–µ–æ—Ä–µ—Ç–∏—á–µ—Å–∫–∏ –≤—Å–µ –æ–Ω–∏ –º–æ–≥—É—Ç –±—ã—Ç—å –¥–æ–±–∞–≤–ª–µ–Ω—ã –∫ –ª—é–±–æ–º—É —ç–ª–µ–º–µ–Ω—Ç—É HTML, –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –≥–ª–æ–±–∞–ª—å–Ω—ã–µ –∞—Ç—Ä–∏–±—É—Ç—ã –Ω–µ –∏–º–µ—é—Ç —ç—Ñ—Ñ–µ–∫—Ç–∞ –ø—Ä–∏ —É—Å—Ç–∞–Ω–æ–≤–∫–µ –Ω–∞ –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã; –Ω–∞–ø—Ä–∏–º–µ—Ä, —É—Å—Ç–∞–Ω–æ–≤–∫–∞ `hidden` –Ω–∞ `<meta>` –Ω–µ –æ—Ç–æ–±—Ä–∞–∂–∞–µ—Ç –º–µ—Ç–∞-—Å–æ–¥–µ—Ä–∂–∏–º–æ–µ.
